How to Use

ScrollKing provides automatic continuous scrolling in any direction:

  1. Click an arrow button to start auto-scrolling in that direction
  2. Click the play/pause button (middle) to pause or resume scrolling
  3. Press Esc key to immediately stop scrolling
  4. Adjust the scroll speed using the slider below
